# ROS2 Install Guide (Jazzy)

## Official Documentation
You can follow the docs, or click the link below. [ROS2 Installation](https://docs.ros.org/en/jazzy/Installation.html)

## Installation Guide 
Use the markdown guide [here](ros2_setup_guide_V1.0.md) to install ROS2 and setup a ROS2 connection with the iRobot Create3 - you don't have to use the create3
but if you are taking robotics 2, it will be good to have it set up.  

## FANUC CRX10 Setup Guide
1. Visit the [Fanuc CRX10 Setup Guide](../fanuc_instructions.md) get the code to run on the robots.

2. Clone the github repo for the FANUC's CRX10 code to your computer.

3. Follow instructions from the repo to get running.
